# PRECISE_MG1655
PRECISE database for wild-type Escherichia coli MG1655

The data and code provided here were used to create and analyze the iModulon structure of the wild-type model strain E. coli MG1655. 

Associated publication: 
Heera Bajpe, Jongoh Shin, Ying Hefner, Richard Szubin, Jaemin Sung, Yuan Yuan, Bernhard O Palsson, The transcriptional regulatory network of the Escherichia coli MG1655 reference strain, Nucleic Acids Research, Volume 54, Issue 3, 10 February 2026, gkag059, https://doi.org/10.1093/nar/gkag059
